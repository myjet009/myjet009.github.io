## ☞ 아래 몇 개만 적어봄

#### - Rust 버전 확인  
```
> rustc --version
```

#### - Rust Update

```
> rustup --update

6주마다 Stable Version 나온다고 함...
```

#### - Rust & Cargo Manual
```
> rustup doc

The Rust Programming Language and The Cargo Book
```
